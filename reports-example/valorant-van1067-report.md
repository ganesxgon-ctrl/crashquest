## Summary

VAN 1067 is often reported when Riot Vanguard cannot keep its required service or platform checks stable while Valorant is running.

## Category

Anti-cheat service or platform integrity

## Likely Causes

- The Vanguard service is stopped, blocked, or did not initialize correctly.
- A pending Windows update or driver update requires a restart.
- Virtualization, firmware, or Windows security settings changed recently.
- Conflicting background software is interfering with Vanguard.

## Safe Fixes

1. Restart the PC, then open Valorant without launching unnecessary background tools.
2. Check that the Riot Vanguard tray icon appears and that the vgc service is running.
3. Install Windows updates, GPU driver updates from the vendor, and restart.
4. Repair or reinstall Riot Vanguard using Riot's official client flow, then restart.
5. Review recently installed overlays, macro tools, or low-level system utilities and temporarily close them before launching the game.
6. Check Riot's service status page for active incidents before making deeper system changes.

## Risky Fixes To Avoid

- Do not install unofficial Vanguard builds.
- Do not run unknown scripts that modify services or registry keys.
- Do not disable core Windows security features permanently.
- Do not reinstall Windows as an early troubleshooting step.
