## Summary

Minecraft Java exit code 1 is a generic crash result. It often points to a Java mismatch, incompatible mods, graphics driver issues, or a broken launcher profile.

## Category

Java runtime, mods, or launcher configuration

## Likely Causes

- A mod or mod loader version does not match the Minecraft version.
- The selected Java runtime is incompatible with the Minecraft version.
- Graphics drivers are outdated or unstable.
- Launcher arguments or allocated memory settings are invalid.

## Safe Fixes

1. Launch the same Minecraft version with no mods to confirm whether the base game starts.
2. Update the Minecraft Launcher and use the bundled Java runtime unless a modpack requires a specific official Java version.
3. Move mods out of the mods folder, then add them back in small groups to isolate the failing mod.
4. Check that Forge, Fabric, Quilt, and mod versions match the Minecraft version.
5. Update GPU drivers from NVIDIA, AMD, Intel, or the device manufacturer's official website.
6. Reset custom JVM arguments in the launcher profile.
