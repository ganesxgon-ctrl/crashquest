# Roadmap

## MVP

- Local JSON error database.
- Game and error-code normalization.
- Markdown report generation.
- Safety rules included in every report.
- Manual research queries for official, Reddit, YouTube, and Google searches.

## Near Term

- Add tests for normalization, database lookup, and report generation.
- Add JSON schema validation for database entries.
- Add a `list` command for supported games and codes.
- Add a `--output` option for custom report directories.
- Add more entries for Riot, Steam, Minecraft, Epic Games, Ubisoft Connect, and Windows runtime errors.

## Later

- Optional live research mode with source ranking.
- Citations and confidence levels per fix.
- Community review workflow for database changes.
- Redaction helpers for logs.
- Machine-readable report output.
