# Source Policy

CrashQuest is designed to produce cautious troubleshooting guidance. Database contributors should keep source reliability in mind even before live web search exists.

## Preferred Sources

1. Official game publisher support pages.
2. Official platform documentation from Microsoft, Steam, Mojang, Riot, GPU vendors, motherboard vendors, or laptop vendors.
3. Official service status pages.
4. Community posts with clear reproduction steps, multiple confirmations, and reversible fixes.
5. Videos that demonstrate official settings or reversible steps.

## Untrusted Sources

Avoid relying on:

- Random DLL download sites.
- Unknown registry patches.
- Reuploaded anti-cheat or launcher files.
- Credential collection forms.
- Fixer executables from unfamiliar domains.
- Posts that recommend irreversible changes without explanation.

## MVP Behavior

The MVP does not fetch live sources. It stores search queries that can be used later by a human contributor or a future research mode.
